## Codebase Patterns
- **Cart Format:** Always use Cart v2 format with `{ version: 2, items: [], updatedAt: string }`
- **Sanity Images:** Use `coalesce(mainImage.asset->url, image.asset->url)` in GROQ queries
- **Backend Enums:** UPPERCASE only (`USER`, `BUYER`, `GROWER`, `ADMIN`)
- **Auth Tokens:** `setAuthToken()` is client-side only - check for SSR before calling
- **Route Protection:** Use `src/proxy.ts` (Next.js 16 renamed from middleware.ts)
- **API Routing:** Email endpoints route based on `NEXT_PUBLIC_EMAIL_SERVICE_ENV`
- **Test Guards:** Long-running intervals should be disabled in test environment by default. Use opt-in flags: `global.__ENABLE_COOKIE_POLLING_IN_TESTS` and `global.__ENABLE_REALTIME_IN_TESTS`.
- **Hero Carousel Heights:** Use reduced heights for better UX - loading/error: h-[300px] md:h-[350px] lg:h-[400px], content: min-h-[400px] md:min-h-[500px] lg:min-h-[600px]
- **Webhook Idempotency:** Use bounded in-memory Set (MAX_PROCESSED_EVENTS=1000) for event deduplication. Evicts oldest on overflow.
- **Webhook Logging:** Structured logger with [WEBHOOK][LEVEL] prefix, always includes eventId, eventType, orderId context.
