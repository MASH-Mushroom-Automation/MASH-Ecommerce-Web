AI MODEL SWITCHING GUIDE
===============================================================================

To change the AI model used by the chatbot:

1. Edit .env file:
   NEXT_PUBLIC_GEMINI_MODEL=gemini-pro  (or any Gemini model)
   
2. Restart dev server:
   Ctrl+C (stop)
   npm run dev (restart)

3. Test with chatbot:
   Open http://localhost:3000
   Click chatbot button
   Ask questions

Available Gemini models:
- gemini-2.0-flash (current, fastest)
- gemini-pro (balanced)
- gemini-pro-vision (images support)
- gemini-ultra (most powerful)
